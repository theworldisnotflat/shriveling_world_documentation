Interface: IComplexAlphaItem

An item grouping for a fixed year and a fixed origin city datas to generate a complex cone the slope for road and slope for each destination city (clock) using a terrestrial transport.

Defined in src/definitions/project.ts:81

This property represents the slope of the road transport for the considered year.

tab

• tab: Array<{ alpha: number ; clock: number }>

Defined in src/definitions/project.ts:88

This property lists for the considered year and the considered origin city each destination city using a terrestrial transport. Each item of this array is a clock in direction of the destination city and a slope corresponding to the transport speed linking the two cities. This array can have zero item.
